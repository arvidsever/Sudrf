# B5 — корректная кассационная подсудность ЯНАО

- КСОЮ и АСОЮ теперь определяются по явному коду субъекта РФ: Ямало-Ненецкий
  автономный округ направляется в Седьмой кассационный суд.
- CLI-команды маршрутизации и загрузки судов принимают код субъекта вместо
  неточного поиска по названию региона.
