# Low-severity hardening sweep

- Исправлено смешивание событий разных раундов производства ВС РФ и выбор статуса по материалу вместо исхода дела.
- Усилены проверки дат, разбор номера карточки, счётчиков выдачи, таблиц и ссылок мировых судей.
- Отмена сетевого запроса больше не удерживает лишний слот троттлинга; CLI строго проверяет звено суда.
- Исправлены маршрутизация гарнизонных военных судов и несколько редких состояний интерфейса мониторинга.
