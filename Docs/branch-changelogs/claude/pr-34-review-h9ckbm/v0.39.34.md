# v0.39.34 — follow-up к ревью PR #34 (восстановление цепочек)

- `CaseOriginResolver.firstCartoteka` поддерживает якорь «Жалобы по делам об АП»
  (`admj`, номера 12-…): первая инстанция ищется в картотеке `adm` мирового или
  районного суда. Ранее такой якорь помечался как `unsupportedCourt` и запись
  не восстанавливалась.
- Убран бессмысленный тернарник выбора уровня суда в
  `CaseOriginResolver.resolveCourt` (обе ветки давали `.district`); поведение
  не изменилось, гарнизонные суды остаются районным уровнем с военной
  вертикалью через `branch`.
