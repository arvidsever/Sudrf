# Изменения — Alpha 0.38.1

## Исправлено

- Минимальная версия TLS для `sudrf.ru` и `msudrf.ru` повышена до TLS 1.2,
  поэтому Xcode больше не предупреждает об устаревшем TLS 1.0. Мировые судьи
  остаются в списке доверенных доменов.
- Устранены предупреждения SwiftUI о сложении `Text`: строка участника дела со
  статьями теперь использует интерполяцию `Text`.
- События календаря получили устойчивые уникальные идентификаторы. Повторные
  заседания и сроки больше не вызывают предупреждения `ForEach` о дубликатах.

## Изменено

- Разрешение на системные уведомления запрашивается только при появлении нового
  события ленты, а не при запуске приложения.
- Обновление бейджа Dock пропускает повторную запись неизменившегося значения.
  Это уменьшает системный шум в консоли, сохраняя уведомления и счётчик новых
  дел.

## Тесты

Полный прогон `swift test`, регенерация Xcode-проекта и Debug-сборка схемы
`Sudrf` для macOS.
