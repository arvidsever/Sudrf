# v0.39.30 — Transport and resolver hardening (draft)

> Branch draft. Final release number, `project.yml`, `Scripts/make-app.sh` and
> `changelog/changelog-v*.md` are updated only during merge/release.

## Fixed

- A stray undefined `0x98` byte no longer makes an otherwise valid cp1251 court
  response undecodable.
- Search now tries the remaining known URL variants after a transport failure;
  cancellation still stops immediately.
- Parallel VSRF requests reserve their throttle slots atomically and are spaced
  at the configured interval.
- A nationwide district-court harvest loads and preserves the persisted cache
  before adding its results.

## Validation

- Added regressions for cp1251 repair, variant fallback, concurrent VSRF
  throttling, district cache preservation, and pre-existing magistrate cache
  loading.
