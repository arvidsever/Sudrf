## Архитектура модели (по описанию друга)

```
input 100×30 RGB
  → CIImage, рендер в 100×30 RGBA пикселей
  → бинаризация: ink iff distance(RGB, (2,103,154)) < 80
  → 100×30 binary mask
  → downsample 100×30 → 64×20 (box averaging)
  → MLMultiArray shape [1, 20, 64]  (channel-first, или [1, 64, 20, 1] в CoreML)
  → MLModel.prediction
  → 5 × MLMultiArray (10 float каждый, softmax)
  → argmax per head → 5 цифр
  → CaptchaAttempt(value: 5digits, confidence: min(softmax[0..4]))
```

Conf = min(softmax по 5 головам) — мера «уверенности» модели.
Top-3 candidates diagnostic показывает топ-N строк по этой мере.

## Архитектура (по описанию друга)

Вход: 100×30 RGB → бинарная маска «чернил» (порог по цвету
~(2, 103, 154)) → downsample 100×30 → 64×20 (box averaging).
Выход: `[1, 5, 10]` — 5 softmax-голов по одной на позицию цифры.

```
input 64×20×1 binary ink mask
  → conv 3×3, 8 filters, leakyReLU, maxpool 2×2
  → conv 3×3, 16 filters, leakyReLU, maxpool 2×2
  → dense 64, leakyReLU
  → 5 × softmax(10)  (5 позиций × 10 цифр)
```

Обучение: mini-batch SGD (24), lr=0.02, ×0.5 на эпохах 10/16/22,
momentum 0.9, L2=1e-4, ранняя остановка. Friend reports 95.1%
per-digit на 4039 captchas.
