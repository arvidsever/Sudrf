# Изменения — Alpha 0.37.1

## Исправлено

Исправлена сборка движения для КАС-апелляций на старом VNKOD-интерфейсе судов
субъектов. Самарский областной суд публикует дела `33а-…` в общей картотеке
«Гражданские и административные дела», поэтому поиск по УИД теперь сначала
пробует эту картотеку и только затем старый КАС-вариант.

Для VNKOD-карточек дополнительно разбираются докладчик, поле «Решение» и текст
судебного акта из вкладок `Document`.

## Тесты

Добавлены регрессионные фикстуры Самарского областного суда: выдача по УИД и
карточка апелляционного определения по делу `33а-647/2026`.

Таргетный прогон `swift test --filter SudrfKitTests`: 179 тестов, 0 падений.
