# Изменения — Alpha 0.9.0

## Симптом

Дело об АП, где суд — **1-я инстанция по существу** (полиция возбудила,
постановление вынес судья), с многократным возвратом на новое рассмотрение,
показывало все круги горсуда, но **ни одного круга ВС**. Реальное дело
(УИД `11RS0001-01-2020-015447-54`, Сыктывкарский горсуд, картотека «Дела об АП,
1-я инстанция»):

| № | Суд | Судья | Результат |
|---|---|---|---|
| 5-45/2021 | горсуд (1-я) | Леконцев | постановление о наказании |
| 12-127/2021 | ВС РК | Колесникова | отменено, новое рассмотрение |
| 5-1469/2021 | горсуд (1-я) | Новикова | производство прекращено |
| 12-283/2021 | ВС РК | Соболев | отменено, новое рассмотрение |
| 5-2418/2021 | горсуд (1-я) | Дульцева | постановление о наказании |
| 12-355/2021 | ВС РК | Попов | оставлено без изменения |

## Причина

v8 для базовой картотеки `adm` (1-я инстанция по существу) отправлял поиск ВС в
картотеку `adm2` («жалобы на решения по жалобам», `delo_id=1513001`). Но жалоба на
**не вступившее в силу постановление судьи** (гл. 30 КоАП) рассматривается судом
субъекта в **другой** картотеке — «жалобы на постановления» (`adm1_case`,
`delo_id=1502001`), номера «12-XXX/ГГГГ». Поиск по adm2 возвращал пусто → ВС не
находился. (`adm2` — это вторая ступень для дел, где 1-й инстанцией по жалобе был
**райсуд**, т.е. база `admj`.)

Иными словами, две ветки КоАП требуют разных вышестоящих картотек, а v8 сводил их
к одной.

## Исправлено

- **Картотека ВС `adm1`.** В `CartotekaRegistry.subject` добавлена «АП: жалобы на
  постановления» (`delo_id=1502001`, `new=0`, `adm1_case`). Та же таблица, что у
  районной «Жалобы по делам об АП», но у суда субъекта.
- **Разделён маппинг** `higherCartotekaIDs` на уровне субъекта:
  - база `adm` (постановление судьи 1-й инстанции) → ВС `adm1` («жалобы на
    постановления»);
  - база `admj` (решение райсуда по жалобе на несудебное постановление) → ВС
    `adm2` («жалобы на решения по жалобам»).

Переопрос домашнего суда (3 круга горсуда) и строго хронологическая сортировка
из v8 уже работали — после починки картотеки шесть инстанций сами встают в
хронологию: 5-45 → 12-127 → 5-1469 → 12-283 → 5-2418 → 12-355.

Классификация ВС-кругов как «круг» (а не ЧЖ) проходит штатным фолбэком по
результату («Отменено с возвращением…», «Оставлено без изменения» — не
«определенческие»). Заголовок ВС-акта — «Решение» (уже из v8).
