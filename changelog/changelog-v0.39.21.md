# A4b — защитный гард маршрутизации CoreML

## Исправлено

- CoreML-ответ для числовой captcha теперь передаётся дальше только при
  достаточной уверенности и формате из пяти ASCII-цифр.
- При низкой уверенности или несовместимом формате используется Vision,
  чтобы не отправлять на судрф уверенный, но заведомо неподходящий ответ.

## Проверка

- Добавлены тесты fallback для low-confidence и несовместимого результата,
  а также регрессия для валидного пятизначного CoreML-ответа.
