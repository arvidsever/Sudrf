# 0.39.32 — Отслеживание и корректность сроков

> Alpha, build 76.

## Исправлено

- Снятие дела с отслеживания использует его стабильный ключ записи, поэтому
  одинаковые номера дел из разных судов больше не могут удалить друг друга.
- Срок кассации рассчитывается только при наличии фактической даты вступления
  в силу или даты апелляции.
- Делегат уведомлений восстанавливается при запуске приложения без запроса
  разрешений.

## Проверка

- Точечные тесты SudrfApp.
