# A8 — чтение дат manifest корпуса

## Исправлено

- `CorpusStore` теперь декодирует ISO-8601 даты из собственного
  `manifest.json` при повторном открытии корпуса.
- Счётчики и даты последнего обучения больше не сбрасываются после
  первой записи manifest.

## Проверка

- Добавлен регрессионный тест: markTrained → flush → reopen сохраняет
  даты, счётчики и pending-значения для обоих видов captcha.
