# Изменения — Alpha 0.19.0

## Вторая кассация: Верховный Суд РФ (vsrf.ru)

В цепочку движения дела добавлена **вторая кассация — Судебные коллегии
ВС РФ**. Верховный Суд живёт на отдельной платформе `vsrf.ru` (не
sud_delo/ГАС «Правосудие»): ответы в UTF-8, капчи нет, поиск — GET-запросом
на `/lk/practice/claims`. Интеграция по хэндоффу из Claude Chat, разметка
выверена по реальным HTML-фикстурам (дело Воробьёва, Республика Коми).

### Новое в SudrfKit

- `VSRFCard.swift` — модели производств ВС (`VSRFProduction`: жалоба «21-…» /
  дело «12-…»), единый парсер карточки и выдачи (`VSRFCardParser` /
  `VSRFSearchParser` на общем каркасе `vs-items-separate` → `vs-item-detail`),
  ключ сквозной привязки `VSRFLinkKey`: по УИД, а при его отсутствии — по
  тройке (суд 1-й инст. + № дела 1-й инст. + фамилия заявителя, с
  нормализацией ё→е и формата ФИО).
- `VSRFClient.swift` — актор-клиент vsrf.ru (троттлинг, ретраи с бэкоффом,
  UTF-8). TLS — обычная проверка Apple; опция `trustVSRFCertificate: true`
  (только для vsrf.ru) на случай сертификата на корнях Минцифры.
- `MovementService`: новый опциональный параметр `vsrf:` (протокол
  `VSRFProviding`; `nil` — прежнее поведение, обратная совместимость).
  После опроса вышестоящих судов сервис ищет производства ВС по УИД и по
  тройке (выдача по одному № дела мешает регионы — отбор строгий).
  Слияние «жалоба → дело»: у истребованного дела событие «Истребовано дело»
  вливается в движение, сама жалоба не дублируется; «отказные»/«возвратные»
  жалобы идут отдельными инстанциями с пометкой.
- `CaseInstance.Level`: новый уровень `vsCassation` между `cassation` (КСОЮ)
  и `supervisory` (Президиум); акт — «Определение Верховного Суда РФ».
- `CaseInstance.note` — пометка инстанции («отказ в передаче», «возврат без
  рассмотрения», «жалоба отклонена»).

### Приложение

- Клетка инстанции ВС: тёмно-красный тинт, чип пометки рядом с «по УИД».
- Стадия «кассация» в Обзоре учитывает и вторую кассацию.
- `VSRFClient` внедрён в живой поиск (`SearchModel`) и в фоновое обновление
  мониторинга (`RefreshCenter`).

### Границы

Тексты судебных актов ВС в панель «Судебные акты» не заводятся — vsrf.ru не
отдаёт текст акта; по ВС показываются движение и результат инстанции.
Надзор (Президиум ВС РФ) в этой версии не реализуется.

### Тесты

`VSRFCardParserTests` + `VSRFMovementTests` на 4 реальных фикстурах
(карточка Воробьёва и три страницы выдачи): разбор карточки/выдачи, отбор
тройкой среди 10 одноимённых дел разных регионов, слияние жалобы в дело,
обратная совместимость без внедрённого клиента. `swift build` — чисто,
`swift test` — 114 тестов зелёные. Живой смоук: поиск по УИД на vsrf.ru
возвращает дело «3-КГ23-1-К3» — совпадает с фикстурами.
